# AirBridge

A high-performance offline file transfer web application using animated QR codes and ultrasonic audio.

## Acknowledgements and Open Source Libraries

This project is made possible by the following incredible open-source projects:

- **[ZBar WASM](https://github.com/undecaf/zbar-wasm)** (LGPL) - High-performance barcode scanning port for WebAssembly.
- **[BC-UR](https://github.com/BlockchainCommons/bc-ur)** (BSD) - Fountain code implementation for uniform resources (Blockchain Commons).
- **[Pako](https://github.com/nodeca/pako)** (MIT) - Zlib port for high-speed compression in JavaScript.
- **[QRCode.js](https://github.com/davidshimjs/qrcodejs)** (MIT) - Cross-browser QR Code generation.
- **[ggwave](https://github.com/ggerganov/ggwave)** (MIT) - Acoustic data transfer.
